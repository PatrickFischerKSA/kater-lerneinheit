# Der gestiefelte Kater - Interaktive Lerneinheit

Diese statische Website ist direkt fur **GitHub Pages** geeignet.

## Upload auf GitHub

1. Neues Repository anlegen.
2. `index.html`, diese `README.md` und den kompletten Ordner `assets/` in das Repository hochladen.
3. In GitHub unter **Settings > Pages** als Source den Branch auswahlen.
4. Danach wird die Seite als Website veroffentlicht.

## Inhalt

- 10 interaktive Multiple-Choice-Fragen
- Prominent eingebundener Podcast mit integriertem Audio-Player
- Begleitdokument als direkt verlinkte Zusatzressource
- Grimm- und Tieck-Teil
- Ausfuhrliche Musterlosungen
- Soundeffekte fur richtig, falsch und das Happy End
- Responsive Layout fur Laptop, Tablet und Smartphone

## Hinweis

Die Soundeffekte sind direkt im Browser per Web Audio API erzeugt. Es werden keine externen Audiodateien oder Bibliotheken benotigt.
